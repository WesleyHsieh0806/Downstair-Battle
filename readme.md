## Usage

1. Compile the file

  ```sh
  g++ -I include/SDL2 main.cpp -o main -lmingw32 -lSDL2main -lSDL2 -lSDL2_mixer -lSDL2_image -L lib 
  ```

2. Click the .exe (main.exe)
